# Task 032 Evidence

이 디렉터리는 Task 032 repository baseline/governance reconciliation의 로컬 실행 증거를 보관한다.

증거 범위:

- Git fetch/status/HEAD/origin-main/start SHA/merge-base/ahead-behind/diff-stat
- `origin/main` 병합 stdout/stderr/exit code
- main-only 두 커밋 log와 net diff
- JSON parse 결과
- adapter validator CLI 결과
- adapter validator unittest 결과
- local workspace adapter unittest 결과
- Python 버전
- 최종 git status/ahead-behind/HEAD

이 증거는 기능 동작 검증이나 Hancom COM 실행 증거가 아니다. Task 032는 기능 개발을 수행하지 않으며, Stage 2 전환이나 최종 HWPX core selection을 선언하지 않는다.
